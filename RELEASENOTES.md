# What's new in LingView 1.0?

## 1.0 (May 8, 2026)
- Offline build process is now functional with two fixed lines in the terminal.
- The display of the transcription has been corrected to properly account for the window width.
- Transcriptions are now wrapped: no more horizontal scrolling!
- Glossing explanations are now displayed in a dedicated sidebar and are much easier to see in the gloss line.
