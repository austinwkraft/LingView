# What's new in LingView 1.1?

## 1.1 (July 7, 2026)
- Major updates to the Search page functionality!
- Added filter options for speaker, dialect, and media duration
- Dialect categories can be updated by editing the dialect_filter.json file
- All fiters are now under a show/hide toggle
- Number and range of search results being shown on the page are now displayed at the top

## 1.0 (May 8, 2026)
- Offline build process is now functional with two fixed lines in the terminal.
- The display of the transcription has been corrected to properly account for the window width.
- Transcriptions are now wrapped: no more horizontal scrolling!
- Glossing explanations are now displayed in a dedicated sidebar and are much easier to see in the gloss line.
